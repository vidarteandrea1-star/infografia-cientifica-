# Infografía Científica: Crianza, cuidado y calidad del agua en la trucha arcoíris

## Descripción del proyecto

Este proyecto consiste en una infografía web interactiva desarrollada en HTML, CSS y JavaScript puro, basada en el estudio científico sobre la crianza, cuidado y calidad del agua en la trucha arcoíris (Oncorhynchus mykiss).

El objetivo principal es presentar de manera visual, clara y moderna los principales datos estadísticos relacionados con temperatura, alimentación, reproducción, sistemas RAS, sanidad y sostenibilidad en la producción acuícola.

La infografía fue desarrollada aplicando técnicas de ingeniería de prompts mediante inteligencia artificial utilizando Z.ai.
